оконная функция - функция агрегации (как group by), которая работает с выделенным набором строк и выполняет вычисление для этого набора в отдельном столбце.

вид: select ..., func(field) over (partition by field)

партиции - разделение по полю столбца, т.е оконная функция по полю имя содержащее в столбце 2 имени и 4 записи, будет разделено по имени.
|--------------|
| имя | задача |
|--------------|
|маша |   1    | - маша 1 партиция
|маша |   3    |
|дима |   2    | - дима 2 партиция
|дима |   4    |
|--------------|

классы оконных функций: (все функции выполняюся в пределах партиции)
	1) aggregate: avg, min, max, sum, count 

	2) ranking: (обязательно указывать order by в партиции (partition by func_field order by any_field))
		dense_rank - (ранг (1 = высшее значение по полю сортировки) внутри партиции), 
		rank - (если одинаковый ранг, то следующий ранг пересткочит повторяющееся кол-во строк), 
		ntile(num) - (разделяет партицию на группы по числу num), 
		cume_dist - ,  
		row_number 

	3) value: (обращение в выделенной партиции к определенному значению по порядку сортировки, обязательно указывать order by в партиции)
		first_value - первое, last_value - последнее, lag - предыдущее, lead - следующее, nth_value - определенное значение

window - создать alias оконной функции в запросе
