index scan: сканирование по индексу, получение из таблицы. (при получении маленького количества данных из общего числа)

index only scan: сканирование и получение по индексу. (при получении маленького количества данных из общего числа)

bitmap scan: сканирование по страницам. (при получении большого количества данных из общего числа)

sequential scan: последовательное сканирование по таблице (при слабой селективности (при сложном условии использущее несколько стоблцов и тд))
